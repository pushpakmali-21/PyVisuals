# PyVisuals

PyVisuals is a browser-based interactive learning environment and IDE designed for data science visualization with Python. It allows users to write and execute Python code directly in their browser without any backend server requirements, focusing on libraries like NumPy, Pandas, and Matplotlib.

## Features

- **In-Browser Python Execution**: Powered by [Pyodide](https://pyodide.org/), running Python entirely on the client side.
- **Interactive Code Editor**: Integrated [Monaco Editor](https://microsoft.github.io/monaco-editor/) providing a VS Code-like experience with syntax highlighting and auto-indentation.
- **Real-time Visualization**: Instantly view plots, charts, and data structures generated by your code.
- **Interactive Modules**: Step-by-step learning modules for popular data science libraries (e.g., NumPy).
- **Responsive Layout**: Resizable split panels for adjusting the editor, visualizer, and terminal workspaces to your preference.
- **Modern UI**: Built with Next.js, React, and styled with Tailwind CSS for a sleek, dark-themed experience.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **UI Library**: [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Editor**: `@monaco-editor/react`
- **Python Runtime**: `pyodide`
- **Icons**: `lucide-react`

## Getting Started

First, run the development server:

```bash

# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Next.js App Router pages and layouts.
- `components/`: Reusable React components.
  - `home/`: Components specific to the landing page.
  - `ide/`: Components for the interactive code environment (Editor, Terminal, Visualizer, Splitter).
  - `modules/`: Components for learning modules (Sidebar, Content, Sandbox).
- `lib/`: Utility functions and types, including the Pyodide runner (`pyodideRunner.ts`).
